HTML5 e CSS

HTML5

O HTML5 é a versão mais recente da linguagem de marcação HTML.

Elemento semântico HTML5 Usado? Descrição
<header> ✔️ Usado para representar o cabeçalho de uma seção ou página.
<nav> ✔️ Usado para representar uma seção de navegação.
<section> ✔️ Usado para representar uma seção de conteúdo tematicamente relacionado.
<article> ✔️ Usado para representar um artigo ou conteúdo independente.
<aside> ✔️ Usado para representar conteúdo relacionado, mas não essencial, como uma barra lateral.
<footer> ✔️ Usado para representar o rodapé de uma seção ou página.

CSS

O CSS (Cascading Style Sheets) é uma linguagem de folhas de estilo utilizada para descrever a apresentação de um documento HTML.

Tipos de seletores CSS Usado? Descrição
Seletor de tipo ✔️ Seleciona elementos com base no nome da tag.
Seletor de classe ✔️ Seleciona elementos com base no valor do atributo "class".
Seletor de ID ✔️ Seleciona um único elemento com base no valor do atributo "id".
Seletor de descendência ✔️ Seleciona elementos que são descendentes de outros elementos.
Seletor de filho direto ✔️ Seleciona elementos que são filhos diretos de outros elementos.
Seletor de irmão adjacente ✔️ Seleciona o elemento irmão imediatamente seguinte de outro elemento.
Seletor de irmão geral ✔️ Seleciona todos os elementos irmãos de outro elemento.
Combinadores de seletores CSS Usado? Descrição
Seletor de descendência (espaço) ✔️ Seleciona elementos que são descendentes de outros elementos.
Seletor de filho direto (>) ✔️ Seleciona elementos que são filhos diretos de outros elementos.
Seletor de irmão adjacente (+) ❌ Seleciona o elemento irmão imediatamente seguinte de outro elemento.
Seletor de irmão geral (~) ❌ Seleciona todos os elementos irmãos de outro elemento.
